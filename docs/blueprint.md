# **App Name**: Jal rakshak

## Core Features:

- Symptom Entry: Form-based entry for recording patient symptoms, severity, and duration.
- Water Source Reporting: Log details about water sources with visual inspection notes and photos.
- GPS Tagging: Automatically capture location of health reports and water sources.
- Offline Data Sync: Robust background service to sync data when internet connectivity is available.
- Alert System: A tool that presents Real-time push notifications for outbreaks and advisories.
- Educational Modules: Offline access to hygiene practices and disease prevention content in multiple languages.
- Multilingual Support: Support for multiple languages via user settings.

## Style Guidelines:

- Primary color: Saturated teal (#4DB6AC) to evoke a sense of trust, health, and safety.
- Background color: Desaturated teal (#F0F4F3) for a clean and calm backdrop.
- Accent color: Analogous blue (#4DD0E1) for highlighting important actions and information.
- Headline font: 'Space Grotesk', a sans-serif font with a techy feel, to suggest forward-thinking health practices. Body font: 'Inter', a neutral sans-serif font.
- Use clear, simple icons to represent health symptoms, water sources, and alert types.
- Design a clear and intuitive form layout for data collection with consistent spacing and alignment.
- Subtle animations for data synchronization status to provide user feedback.
